# pc_calendar

A local calendar application built with React, Vite, and Electron. It features a Google Calendar-like look and feel, allowing you to interactively create, edit, and delete events. Data is persisted locally using `electron-store`.

## Features
- Month view calendar
- Interactive event creation
- Edit and delete existing events
- Persistent local storage

## Getting Started

```bash
npm install
npm run dev
```
